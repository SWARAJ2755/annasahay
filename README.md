# 🍱 AnnaSahay – School Grain & Meal Register System

AnnaSahay is a **web-based School Grain & Meal Register System** designed to simplify the management of the **PM POSHAN (Mid-Day Meal) Scheme** in government and aided schools.

The application enables schools to efficiently manage daily meal records, monitor student attendance, maintain food grain inventory, generate monthly reports, and print official registers through an intuitive, responsive, and multilingual interface.
