🍱 AnnaSahay – School Grain & Meal Register System
AnnaSahay is a web-based School Grain & Meal Register System developed to simplify the management of the Mid-Day Meal Scheme in government schools. The application enables schools to maintain daily meal records, manage food grain inventory, monitor student attendance, generate reports, and print official registers through an intuitive and user-friendly interface.
